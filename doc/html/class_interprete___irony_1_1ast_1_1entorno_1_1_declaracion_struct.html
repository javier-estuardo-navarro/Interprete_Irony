Descripción detallada

La clase DeclaracionStruct es un nodo en el cual se declara un Struct y se agrega al entorno.

Esto es equivalente a instanciar una clase en un lenguaje orientado a objetos.

AST  arbol

El nivel de abstraccion del Arbol Abstracto de Analisis Sintactico requiere de una forma de recorrerlo, ya que todo forma parte de un NodoAST se define el metodo ejecutar() en todos aquellos nodos que sean Instrucciones, al hacer llamadas al mismo se recorre de una forma abstracta nuestro arbol.

Parámetros
entUn entorno actual de ejecucion el cual gestiona todo lo relacionado con la tabla de simbolos.
arbolUn Arbol Abstracto de Analisis Sintactico, el cual esta compuesto por NodosAST los cuales pueden ser Expresiones o Instrucciones.
Devuelve
Devuelve objetos resultado de sintetizar valores implicitos de una Expresion por ejemplo: En una Instruccion While puede haber una Expresion Return que retorna un valor.

Implementa Interprete_Irony.ast.altaAbstraccion.Instruccion.
